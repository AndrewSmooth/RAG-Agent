# Управление основными средствами (Asset Management)

## Общее описание
Модуль управления основными средствами предназначен для учета и контроля всех материальных активов компании: зданий, сооружений, оборудования, транспортных средств и другого имущества. Модуль обеспечивает полный жизненный цикл управления активами от приобретения до списания.

## Жизненный цикл актива
1. **Приобретение**: учет первоначальной стоимости, даты ввода в эксплуатацию, срока полезного использования
2. **Эксплуатация**: начисление амортизации, учет ремонтов и модернизаций, контроль технического состояния
3. **Перемещение**: учет перемещения активов между подразделениями и площадками
4. **Модернизация**: учет улучшений, увеличивающих стоимость или срок службы актива
5. **Списание**: процедура вывода актива из эксплуатации с учетом остаточной стоимости

## Классификация активов
Активы классифицируются по следующим категориям:
- Здания и сооружения
- Машины и оборудование
- Транспортные средства
- Вычислительная техника
- Инвентарь и принадлежности

Каждая категория имеет свои правила начисления амортизации и нормы эксплуатации.

## Начисление амортизации
Система поддерживает различные методы начисления амортизации:
- Линейный метод
- Метод уменьшаемого остатка
- Метод суммы чисел лет срока полезного использования
- Производственный метод (пропорционально объему продукции)

Параметры амортизации устанавливаются для каждой категории активов и могут быть скорректированы для отдельных активов.

## Интеграция с другими модулями
- С финансовым модулем для отражения амортизации в бухгалтерском учете
- С модулем производства для учета износа производственного оборудования
- С модулем ремонтов и обслуживания для планирования ТО
- С модулем закупок для учета приобретения новых активов
- С налоговым модулем для расчета амортизации в целях налогообложения

## Отчетность
- Ведомость основных средств
- Расчет амортизации
- Анализ износа активов
- План-график ремонтов и ТО
- Отчет о движении активов (поступление, перемещение, списание)
