# Table: bom (Bill of Materials)
## Notes
- Спецификация материала (BOM) определяет состав продукта и его компоненты
- Каждая спецификация связана с конкретным продуктом и имеет версию
- Статус 'superseded' означает, что спецификация заменена новой версией
- Стоимость прямых материалов используется для калькуляции себестоимости продукции
- Опасные и ограниченные материалы требуют особого учета и хранения
- Сборочные единицы, сырьевые и упаковочные материалы структурируют состав продукта
- Условия хранения и срок хранения критичны для управления запасами
- Экологическое воздействие и инструкции по утилизации используются для экологического управления
- Вес и объем готового продукта используются для логистики и транспортировки
- Данные о спецификации используются для планирования потребности в материалах (MRP)
