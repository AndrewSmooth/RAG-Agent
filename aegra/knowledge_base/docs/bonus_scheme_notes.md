# Table: bonus_scheme
## Notes
- Схема премирования определяет правила и условия выплаты премий сотрудникам
- Каждая схема действует в определенный период времени (effective_date - expiration_date)
- Веса (weights) должны в сумме давать 1.0 для корректного расчета премии
- Процент отсрочки (deferral_percentage) используется для долгосрочного мотивационного воздействия
- Некоторые премии могут быть частично или полностью не облагаемыми налогом
- Бюджет на премирование должен быть утвержден в рамках общего бюджета компании
- При расчете премии по процентной схеме: bonus = salary * bonus_percentage * performance_factor
- При расчете по многоуровневой схеме (tiered) используются различные ставки в зависимости от результатов
