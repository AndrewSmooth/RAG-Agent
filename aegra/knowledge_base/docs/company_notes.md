# Table: company
## Notes
- Поле `tax_id` должно быть уникальным для каждой компании
- Поле `registration_number` используется для идентификации в государственных реестрах
- Статус `liquidated` означает, что компания официально прекратила свою деятельность
- Размер компании определяется по количеству сотрудников: small (1-50), medium (51-250), large (251+)
- Все финансовые операции привязываются к конкретной компании через поле `company_id`
