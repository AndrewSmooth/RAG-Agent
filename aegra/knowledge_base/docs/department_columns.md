# Table: department
## Columns
- `id`: уникальный идентификатор отдела (первичный ключ)
- `business_unit_id`: ссылка на бизнес-подразделение (внешний ключ)
- `name`: наименование отдела
- `code`: код отдела (уникальный в пределах бизнес-подразделения)
- `description`: описание функций отдела
- `manager_id`: идентификатор руководителя отдела
- `parent_dept_id`: ссылка на вышестоящий отдел (для иерархии)
- `cost_center`: код центра затрат
- `budget`: годовой бюджет отдела
- `currency`: валюта бюджета
- `established_date`: дата создания отдела
- `headcount_plan`: плановая численность сотрудников
- `headcount_current`: текущая численность сотрудников
- `status`: статус отдела (active, inactive, reorganized)
- `created_at`: дата и время создания записи
- `updated_at`: дата и время последнего обновления записи
- `created_by`: идентификатор пользователя, создавшего запись
- `updated_by`: идентификатор пользователя, последним обновившего запись
- `is_active`: флаг активности записи (true/false)
- `version`: версия записи для контроля параллельных изменений
