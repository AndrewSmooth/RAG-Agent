# Table: downtime_reason
## Notes
- Причины простоя используются для классификации и анализа времени неработоспособности оборудования
- Каждая причина имеет уникальный код для использования в системе учета простоев
- Категория простоя помогает группировать причины для анализа и выявления тенденций
- Уровень серьезности и влияние на производство используются для приоритизации улучшений
- Рекомендуемые действия по предотвращению помогают в разработке планов улучшения
- Стоимость простоя в час используется для экономического обоснования инвестиций в улучшения
- Данные о частоте возникновения и количестве случаев за год используются для анализа эффективности мер
- Целевое время устранения используется для оценки эффективности команд по обслуживанию
