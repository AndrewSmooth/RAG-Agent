# Table: employee_training
## Notes
- Таблица связывает сотрудников с конкретными сессиями и программами обучения
- Статус записи `enrollment_status` отражает этап участия сотрудника в обучении
- При изменении статуса с 'waitlisted' на 'registered' сотрудник получает уведомление
- Оценка `grade` может быть числовой (0-100) или буквенной (A, B, C, D, F)
- Сертификат выдается только при успешном прохождении программы (grade >= passing_score)
- Обратная связь от сотрудника и тренера используется для улучшения программ
- Учет расходов ведется по центру затрат и статье бюджета для финансового контроля
- Приобретенные навыки и улучшенные компетенции обновляют профиль сотрудника
- Возмещение расходов возможно для программ, одобренных руководством
