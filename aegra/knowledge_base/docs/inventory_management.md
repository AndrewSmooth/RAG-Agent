# Управление запасами

## Определение
Управление запасами — это процесс планирования, контроля и оптимизации уровней запасов материалов, компонентов и готовой продукции. Цель управления запасами — обеспечить наличие необходимых материалов для производства и удовлетворения спроса при минимизации затрат на хранение.

## Функции управления запасами
- Прогнозирование спроса
- Планирование закупок
- Контроль уровней запасов
- Управление складскими операциями
- Анализ оборачиваемости запасов
- Минимизация потерь и устаревания

## Типы запасов
### 1. Текущие запасы
- Запасы, используемые в текущем производственном цикле
- Обеспечивают непрерывность производства
- Уровень определяется на основе плана производства

### 2. Страховые запасы
- Резервные запасы для защиты от неопределенности
- Компенсируют колебания спроса и поставок
- Уровень определяется на основе анализа рисков

### 3. Сезонные запасы
- Запасы, создаваемые перед периодом повышенного спроса
- Позволяют сгладить производственные нагрузки
- Требуют точного прогнозирования спроса

### 4. Транзитные запасы
- Запасы в пути между поставщиком и потребителем
- Зависят от времени поставки
- Учитываются при планировании закупок

## Методы управления запасами
### 1. Метод фиксированного размера заказа (EOQ)
- Заказы размещаются при достижении точки перезаказа
- Размер заказа постоянный и оптимален по критерию минимума затрат
- Подходит для материалов с постоянным спросом

**Формула EOQ:**
```
EOQ = √(2 × D × S / H)
```
Где:
- D = Годовой спрос
- S = Стоимость размещения заказа
- H = Стоимость хранения единицы запаса в год

### 2. Метод фиксированного интервала заказа
- Заказы размещаются с фиксированным интервалом
- Размер заказа переменный и зависит от текущего уровня запасов
- Подходит для материалов с нерегулярным спросом

### 3. Метод ABC
- Классификация запасов по важности
- Группа A: 20% номенклатуры, 80% стоимости
- Группа B: 30% номенклатуры, 15% стоимости
- Группа C: 50% номенклатуры, 5% стоимости
- Разные методы управления для разных групп

### 4. Метод JIT (Just-In-Time)
- Минимизация запасов
- Поставки "точно в срок"
- Требует надежных поставщиков и точного планирования
- Снижает затраты на хранение, но увеличивает риски

## Точки управления запасами
### 1. Точка перезаказа (Reorder Point)
```
Точка перезаказа = (Средний дневной спрос × Время поставки) + Страховой запас
```

### 2. Максимальный уровень запасов
```
Максимальный уровень = Точка перезаказа + Размер заказа
```

### 3. Минимальный уровень запасов
- Уровень, при котором необходимо срочно принять меры
- Часто совпадает со страховыми запасами

## Применение в системе
Наша ERP-система предоставляет комплексные инструменты для управления запасами:

### 1. Учет запасов
- Централизованный учет в таблице `inventory` (предполагаемая)
- Контроль по номенклатуре, партиям, срокам годности
- Учет в нескольких единицах измерения
- Интеграция с системами штрих-кодирования

### 2. Планирование закупок
- Автоматическое определение потребности в материалах (MRP)
- Генерация заказов на закупку
- Учет времени поставки и минимальных объемов заказа
- Анализ альтернативных поставщиков

### 3. Контроль уровней запасов
- Автоматическое оповещение при достижении точек перезаказа
- Мониторинг оборачиваемости запасов
- Выявление медленноходящих и устаревших запасов
- Анализ структуры запасов по методу ABC

### 4. Складские операции
- Управление складскими зонами и местами хранения
- Оптимизация маршрутов перемещения
- Контроль сроков годности и партий
- Проведение инвентаризаций

## Показатели эффективности управления запасами
### 1. Оборачиваемость запасов
```
Оборачиваемость = Стоимость реализованной продукции / Средний уровень запасов
```
- Показывает, сколько раз обновляются запасы за период
- Чем выше, тем эффективнее управление

### 2. Уровень обслуживания
```
Уровень обслуживания = (Количество удовлетворенных заказов / Общее количество заказов) × 100%
```
- Показывает способность удовлетворять спрос из наличия

### 3. Затраты на хранение
- Процент от стоимости запасов
- Включает аренду, охрану, страхование, устаревание

### 4. Точность складского учета
```
Точность = (Количество совпадений при инвентаризации / Общее количество проверенных позиций) × 100%
```

## Непрерывное улучшение
Система поддерживает постоянную оптимизацию управления запасами:
- Анализ исторических данных для улучшения прогнозов
- Оптимизация точек перезаказа и страховых запасов
- Пересмотр методов управления для разных групп номенклатуры
- Интеграция с поставщиками для совместного планирования
