# Table: maintenance_record
## Notes
- Запись о техническом обслуживании фиксирует фактическое выполнение работ по оборудованию
- Каждая запись связана с конкретным оборудованием через `machine_id`
- Тип обслуживания определяет категорию работ и их приоритет
- Время простоя и трудозатраты используются для анализа эффективности обслуживания
- Замененные запчасти и использованные инструменты учитываются в системе управления запасами
- Выявленные нарушения безопасности анализируются для улучшения процедур
- Проверка качества после работ обязательна для критичного оборудования
- Фактическая стоимость сравнивается с оценочной для контроля бюджета
