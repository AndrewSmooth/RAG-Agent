# Table: operation
## Notes
- Производственная операция представляет собой элементарную технологическую операцию
- Каждая операция имеет уникальный код для использования в маршрутах производства
- Нормативное время используется для планирования и калькуляции себестоимости
- Время наладки и переналадки критично при планировании производства нескольких продуктов
- Требования к персоналу, оборудованию и материалам используются для проверки возможности выполнения
- Контрольные точки качества определяют, когда и что нужно проверять
- Энергопотребление и образование отходов используются для анализа экологического воздействия
- Нормативный процент брака и коэффициент выхода используются для планирования и анализа
- Уровень автоматизации влияет на требования к персоналу и производственную гибкость
