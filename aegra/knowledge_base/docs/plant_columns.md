# Table: plant
## Columns
- `id`: уникальный идентификатор производственного подразделения (первичный ключ)
- `company_id`: ссылка на компанию (внешний ключ)
- `name`: наименование завода/производственного подразделения
- `code`: код завода (уникальный в пределах компании)
- `description`: описание производственных мощностей и специализации
- `address`: юридический адрес завода
- `location`: географические координаты (широта, долгота)
- `manager_id`: идентификатор руководителя завода
- `established_date`: дата ввода в эксплуатацию
- `status`: статус завода (active, inactive, under_construction, modernization)
- `production_capacity_units`: единица измерения производственной мощности
- `production_capacity_value`: значение производственной мощности в единицах
- `shifts_count`: количество смен
- `work_days_per_week`: количество рабочих дней в неделю
- `safety_rating`: рейтинг безопасности (1-10)
- `environmental_rating`: экологический рейтинг (1-10)
- `last_inspection_date`: дата последней проверки
- `next_inspection_date`: дата следующей проверки
- `created_at`: дата и время создания записи
- `updated_at`: дата и время последнего обновления записи
- `created_by`: идентификатор пользователя, создавшего запись
- `updated_by`: идентификатор пользователя, последним обновившего запись
- `is_active`: флаг активности записи (true/false)
- `version`: версия записи для контроля параллельных изменений
