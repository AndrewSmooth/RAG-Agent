# Table: position
## Columns
- `id`: уникальный идентификатор должности (первичный ключ)
- `company_id`: ссылка на компанию (внешний ключ)
- `name`: наименование должности
- `code`: код должности (уникальный в пределах компании)
- `description`: полное описание функций и обязанностей
- `department_type`: тип отдела, к которому относится должность
- `grade`: грейд должности (A1, A2, B1, B2, C1, C2, D1, D2)
- `salary_min`: минимальный оклад для данной должности
- `salary_max`: максимальный оклад для данной должности
- `salary_currency`: валюта оклада
- `fte_required`: количество единиц полной занятости (FTE)
- `reports_to`: ссылка на должность, которой подчиняется данная должность
- `is_management`: флаг руководящей должности
- `is_budgeted`: флаг бюджетной должности
- `status`: статус должности (active, inactive, frozen)
- `created_at`: дата и время создания записи
- `updated_at`: дата и время последнего обновления записи
- `created_by`: идентификатор пользователя, создавшего запись
- `updated_by`: идентификатор пользователя, последним обновившего запись
- `is_active`: флаг активности записи (true/false)
- `version`: версия записи для контроля параллельных изменений
