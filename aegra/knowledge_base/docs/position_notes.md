# Table: position
## Notes
- Каждая должность должна быть привязана к одной компании
- Поле `code` должно быть уникальным в пределах одной компании
- Грейд должности определяет диапазон допустимых окладов
- FTE (Full-Time Equivalent) показывает, сколько полных ставок требуется на данную должность
- Должности с `is_budgeted = true` включаются в бюджетный процесс
- При заморозке вакансий должности получают статус 'frozen', но не удаляются
- Иерархия подчинения строится через поле `reports_to`
- Минимальный и максимальный оклад используются для контроля при назначении зарплаты
