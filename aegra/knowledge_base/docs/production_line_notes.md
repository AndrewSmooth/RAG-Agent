# Table: production_line
## Notes
- Производственная линия представляет собой технологическую цепочку на заводе
- Каждая линия должна быть привязана к одному заводу
- Поле `code` должно быть уникальным в пределах одного завода
- Коэффициент эффективности (OEE) рассчитывается как произведение доступности, производительности и качества
- Время простоя включает плановые и неплановые остановки
- Контрольные точки качества используются для проверки параметров продукции на разных этапах
- Уровень автоматизации влияет на требования к персоналу и техническому обслуживанию
- Энергопотребление используется для анализа производственных затрат и экологического воздействия
