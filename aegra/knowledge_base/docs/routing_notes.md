# Table: routing
## Notes
- Маршрут производства определяет последовательность операций для выпуска продукта
- Каждый маршрут связан с конкретным продуктом и имеет версию
- Статус 'superseded' означает, что маршрут заменен новой версией
- Последовательность рабочих центров и операций определяет технологический процесс
- Параллельные операции позволяют выполнять несколько операций одновременно
- Контрольные точки качества и уровни инспекции определяют, где и как проверять продукцию
- Критические точки безопасности требуют особого внимания при организации производства
- Схема движения материалов и информации используется для оптимизации логистики
- Точки автоматизации и узкие места используются для планирования улучшений производства
- Оценочная продолжительность используется для планирования и калькуляции себестоимости
