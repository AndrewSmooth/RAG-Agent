# Table: salary_structure
## Columns
- `id`: уникальный идентификатор структуры оклада (первичный ключ)
- `company_id`: ссылка на компанию (внешний ключ)
- `grade`: грейд оклада (A1, A2, B1, B2, C1, C2, D1, D2)
- `step`: шаг в грейде (1, 2, 3, 4, 5)
- `base_salary`: базовый оклад для данного грейда и шага
- `salary_currency`: валюта оклада
- `effective_date`: дата вступления в силу данной структуры
- `expiration_date`: дата окончания действия данной структуры
- `region_factor`: региональный коэффициент (для разных городов и регионов)
- `experience_factor_min`: минимальный коэффициент опыта
- `experience_factor_max`: максимальный коэффициент опыта
- `performance_factor_min`: минимальный коэффициент производительности
- `performance_factor_max`: максимальный коэффициент производительности
- `market_rate`: рыночная ставка для данного грейда
- `midpoint`: середина диапазона оклада
- `range_min`: нижняя граница диапазона оклада
- `range_max`: верхняя граница диапазона оклада
- `created_at`: дата и время создания записи
- `updated_at`: дата и время последнего обновления записи
- `created_by`: идентификатор пользователя, создавшего запись
- `updated_by`: идентификатор пользователя, последним обновившего запись
- `is_active`: флаг активности записи (true/false)
- `version`: версия записи для контроля параллельных изменений
