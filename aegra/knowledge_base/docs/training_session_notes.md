# Table: training_session
## Notes
- Сессия обучения представляет собой конкретное проведение программы в определенное время
- Каждая сессия связана с одной программой обучения через `program_id`
- Статус 'planned' означает, что сессия запланирована, но еще не началась
- При достижении максимальной вместимости новые участники попадают в лист ожидания
- Для онлайн-сессий поле `location` содержит URL вебинара или платформы
- После завершения сессии статус меняется на 'completed' и становится доступна оценка
- Расходы на сессию учитываются по центру затрат и статье бюджета
- Тренер может быть внутренним сотрудником или внешним специалистом
- Время проведения указывается в часовом поясе компании для избежания путаницы
