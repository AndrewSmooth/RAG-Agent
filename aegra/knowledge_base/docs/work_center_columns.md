# Table: work_center
## Columns
- `id`: уникальный идентификатор рабочего центра (первичный ключ)
- `production_line_id`: ссылка на производственную линию (внешний ключ)
- `name`: наименование рабочего центра
- `code`: код рабочего центра (уникальный в пределах линии)
- `description`: описание выполняемых операций
- `machine_id`: ссылка на основное оборудование
- `operator_count`: количество операторов
- `shift_pattern`: график работы (например, 2/2, 3/3)
- `status`: статус рабочего центра (active, inactive, maintenance)
- `setup_time_minutes`: время наладки в минутах
- `cycle_time_seconds`: цикловое время в секундах
- `changeover_time_minutes`: время переналадки между продуктами
- `scrap_rate_percent`: процент брака
- `quality_first_pass_rate`: процент продукции, прошедшей контроль с первого раза
- `last_performance_review`: дата последнего анализа производительности
- `next_performance_review`: дата следующего анализа производительности
- `safety_incidents_count`: количество инцидентов за последний год
- `training_required`: требуется ли дополнительное обучение персонала
- `created_at`: дата и время создания записи
- `updated_at`: дата и время последнего обновления записи
- `created_by`: идентификатор пользователя, создавшего запись
- `updated_by`: идентификатор пользователя, последним обновившего запись
- `is_active`: флаг активности записи (true/false)
- `version`: версия записи для контроля параллельных изменений
