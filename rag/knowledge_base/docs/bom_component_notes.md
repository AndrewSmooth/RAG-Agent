# Table: bom_component
## Notes
- Компонент спецификации определяет конкретный материал в составе продукта
- Каждый компонент связан с определенной спецификацией (BOM)
- Коэффициент потерь учитывает нормативные потери при производстве
- Время поставки и минимальное количество заказа используются для планирования закупок
- Условия хранения и срок хранения критичны для управления складом
- Требования к качеству и инспекции определяют, как проверять поступающие материалы
- Прослеживаемость и контроль по партиям требуются для критичных компонентов
- Правила замены определяют, какие альтернативные материалы могут быть использованы
- Данные о стоимости и поставщиках используются для анализа закупок и управления цепочками поставок
