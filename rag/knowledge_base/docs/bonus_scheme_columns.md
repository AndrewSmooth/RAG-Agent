# Table: bonus_scheme
## Columns
- `id`: уникальный идентификатор схемы премирования (первичный ключ)
- `company_id`: ссылка на компанию (внешний ключ)
- `name`: наименование схемы премирования
- `description`: описание условий и критериев премирования
- `type`: тип премии (performance, project, retention, sign-on, referral)
- `calculation_method`: метод расчета (percentage, fixed_amount, tiered)
- `target_group`: целевая группа сотрудников (all, management, sales, production)
- `frequency`: периодичность выплаты (monthly, quarterly, annually)
- `funding_source`: источник финансирования премии
- `budget`: бюджет на премирование
- `budget_currency`: валюта бюджета
- `min_performance_score`: минимальный балл производительности для получения премии
- `max_bonus_percentage`: максимальный процент премии от оклада
- `performance_weight`: вес производительности в расчете премии
- `financial_weight`: вес финансовых результатов в расчете премии
- `team_weight`: вес командных результатов в расчете премии
- `individual_weight`: вес индивидуальных результатов в расчете премии
- `approval_required`: требуется ли утверждение премии руководством
- `taxable`: является ли премия облагаемой налогом
- `deferral_percentage`: процент отсрочки выплаты премии
- `deferral_period`: период отсрочки выплаты премии (в месяцах)
- `effective_date`: дата вступления в силу схемы
- `expiration_date`: дата окончания действия схемы
- `created_at`: дата и время создания записи
- `updated_at`: дата и время последнего обновления записи
- `created_by`: идентификатор пользователя, создавшего запись
- `updated_by`: идентификатор пользователя, последним обновившего запись
- `is_active`: флаг активности записи (true/false)
- `version`: версия записи для контроля параллельных изменений
