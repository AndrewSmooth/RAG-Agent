# Table: business_unit
## Columns
- `id`: уникальный идентификатор бизнес-подразделения (первичный ключ)
- `company_id`: ссылка на компанию (внешний ключ)
- `name`: наименование бизнес-подразделения
- `code`: код подразделения (уникальный в пределах компании)
- `description`: описание деятельности подразделения
- `manager_id`: идентификатор руководителя подразделения
- `parent_unit_id`: ссылка на вышестоящее подразделение (для иерархии)
- `cost_center`: код центра затрат
- `profit_center`: код центра прибыли
- `budget`: годовой бюджет подразделения
- `currency`: валюта бюджета
- `established_date`: дата создания подразделения
- `status`: статус подразделения (active, inactive, merged)
- `created_at`: дата и время создания записи
- `updated_at`: дата и время последнего обновления записи
- `created_by`: идентификатор пользователя, создавшего запись
- `updated_by`: идентификатор пользователя, последним обновившего запись
- `is_active`: флаг активности записи (true/false)
- `version`: версия записи для контроля параллельных изменений
