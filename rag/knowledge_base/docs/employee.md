# Table: employees

## Columns
- `id`: уникальный идентификатор сотрудника
- `name`: полное имя сотрудника
- `age`: возраст, целое число
- `salary`: оклад в рублях, до налогообложения
- `dept_id`: ссылка на отдел (внешний ключ)

## Notes
- Поле `salary` рассчитывается по формуле: base_salary * experience_factor + bonus
- Удалённые сотрудники не удаляются из таблицы, а помечаются в отдельной таблице `employee_status`
