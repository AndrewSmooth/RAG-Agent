# Table: machine
## Notes
- Оборудование представляет собой физические машины и станки на производстве
- Каждое оборудование должно быть привязано к одному рабочему центру
- Серийный номер должен быть уникальным в пределах компании
- Статус 'under_maintenance' означает, что оборудование временно недоступно для производства
- Статус 'decommissioned' означает, что оборудование списано и больше не используется
- Интервал технического обслуживания используется для автоматического планирования ТО
- Уровень критичности влияет на приоритеты при планировании обслуживания и закупке запчастей
- Данные об энергопотреблении и затратах на обслуживание используются для анализа эффективности
