# Table: maintenance_schedule
## Notes
- График технического обслуживания определяет регулярные задачи по обслуживанию оборудования
- Каждая запись связана с конкретным оборудованием через `machine_id`
- Тип графика определяет стратегию обслуживания: профилактическое, предиктивное и т.д.
- Интервал выполнения в днях используется для автоматического расчета дат следующего ТО
- Необходимые навыки, инструменты и запчасти используются для планирования ресурсов
- Приоритет задачи влияет на расстановку при планировании работ
- Допустимое отклонение от графика позволяет учитывать небольшие задержки без нарушения графика
- Данные о стоимости используются для бюджетирования и анализа эффективности обслуживания
