# Table: performance_review
## Notes
- Оценка производительности проводится регулярно (ежеквартально, раз в полгода или ежегодно)
- Общий балл рассчитывается как средневзвешенное по всем критериям оценки
- Уровень производительности определяется по шкале: 1-2 (Poor), 2.1-3 (Below Average), 3.1-4 (Average), 4.1-4.7 (Good), 4.8-5 (Excellent)
- Рекомендации по премированию используются при расчете бонусов по bonus_scheme
- План развития должен содержать конкретные действия, ресурсы и сроки
- Оценки с статусом 'draft' могут редактироваться, после отправки статус меняется на 'submitted'
- Утверждение оценки может требовать согласования с вышестоящим руководством
- История оценок используется для анализа карьерного роста и принятия кадровых решений
