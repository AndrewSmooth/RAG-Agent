# Table: production_line
## Columns
- `id`: уникальный идентификатор производственной линии (первичный ключ)
- `plant_id`: ссылка на завод (внешний ключ)
- `name`: наименование производственной линии
- `code`: код линии (уникальный в пределах завода)
- `description`: описание технологического процесса и оборудования
- `product_type`: тип производимой продукции
- `status`: статус линии (active, inactive, maintenance, upgrade)
- `capacity_per_hour`: производственная мощность в час
- `current_output`: текущий объем производства
- `target_output`: целевой объем производства
- `downtime_minutes`: время простоя в текущем месяце (в минутах)
- `efficiency_rate`: коэффициент эффективности (OEE)
- `last_calibration_date`: дата последней калибровки оборудования
- `next_calibration_date`: дата следующей калибровки
- `safety_interlocks_count`: количество систем аварийной блокировки
- `quality_checkpoints_count`: количество контрольных точек качества
- `automation_level`: уровень автоматизации (low, medium, high, full)
- `energy_consumption_kwh`: энергопотребление в кВт·ч на единицу продукции
- `created_at`: дата и время создания записи
- `updated_at`: дата и время последнего обновления записи
- `created_by`: идентификатор пользователя, создавшего запись
- `updated_by`: идентификатор пользователя, последним обновившего запись
- `is_active`: флаг активности записи (true/false)
- `version`: версия записи для контроля параллельных изменений
