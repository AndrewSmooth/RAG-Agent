# Table: production_order
## Columns
- `id`: уникальный идентификатор производственного заказа (первичный ключ)
- `plant_id`: ссылка на завод (внешний ключ)
- `order_number`: номер заказа (уникальный)
- `product_id`: ссылка на продукт (внешний ключ)
- `product_name`: наименование продукта
- `planned_start_date`: планируемая дата начала производства
- `planned_end_date`: планируемая дата окончания производства
- `actual_start_date`: фактическая дата начала производства
- `actual_end_date`: фактическая дата окончания производства
- `planned_quantity`: планируемое количество единиц
- `produced_quantity`: фактически произведенное количество
- `scrap_quantity`: количество брака
- `status`: статус заказа (planned, released, in_progress, completed, cancelled, on_hold)
- `priority`: приоритет заказа (low, medium, high, rush)
- `routing_id`: ссылка на маршрут производства
- `bom_id`: ссылка на спецификацию материала
- `work_center_id`: ссылка на рабочий центр (если специализированный)
- `production_line_id`: ссылка на производственную линию
- `supervisor_id`: идентификатор руководителя производства
- `quality_plan_id`: ссылка на план контроля качества
- `batch_number`: номер производственной партии
- `expiration_date`: срок годности продукции
- `customer_id`: ссылка на клиента (если заказ по заказу клиента)
- `sales_order_id`: ссылка на заказ клиента
- `cost_center`: центр затрат для учета расходов
- `budget_line`: статья бюджета
- `created_at`: дата и время создания записи
- `updated_at`: дата и время последнего обновления записи
- `created_by`: идентификатор пользователя, создавшего запись
- `updated_by`: идентификатор пользователя, последним обновившего запись
- `is_active`: флаг активности записи (true/false)
- `version`: версия записи для контроля параллельных изменений
