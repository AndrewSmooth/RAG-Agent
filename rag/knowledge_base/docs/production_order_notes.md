# Table: production_order
## Notes
- Производственный заказ представляет собой задание на выпуск определенного количества продукции
- Каждый заказ имеет уникальный номер в пределах завода
- Статус 'released' означает, что заказ передан в производство
- Статус 'on_hold' используется при приостановке производства по различным причинам
- Приоритет заказа влияет на его расстановку в производственном расписании
- Маршрут производства и спецификация материала определяют технологический процесс
- Количество брака используется для анализа качества и расчета выхода годной продукции
- Номер партии и срок годности критичны для отслеживания продукции и управления запасами
- Данные о центре затрат и статье бюджета используются для учета производственных расходов
