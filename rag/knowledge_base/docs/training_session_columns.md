# Table: training_session
## Columns
- `id`: уникальный идентификатор сессии обучения (первичный ключ)
- `program_id`: ссылка на программу обучения (внешний ключ)
- `session_code`: код сессии (уникальный)
- `title`: наименование сессии
- `description`: описание сессии
- `start_date`: дата начала сессии
- `end_date`: дата окончания сессии
- `start_time`: время начала (в часовом поясе компании)
- `end_time`: время окончания (в часовом поясе компании)
- `time_zone`: часовой пояс проведения сессии
- `location`: место проведения (адрес или URL для онлайн)
- `trainer_id`: идентификатор тренера
- `trainer_name`: имя тренера
- `trainer_type`: тип тренера (internal, external)
- `status`: статус сессии (planned, in_progress, completed, cancelled)
- `max_capacity`: максимальная вместимость
- `current_enrollment`: текущее количество записанных
- `waiting_list_count`: количество в листе ожидания
- `materials_url`: URL для доступа к материалам
- `evaluation_form_url`: URL для формы оценки
- `cost_center`: центр затрат для учета расходов
- `budget_line`: статья бюджета
- `created_at`: дата и время создания записи
- `updated_at`: дата и время последнего обновления записи
- `created_by`: идентификатор пользователя, создавшего запись
- `updated_by`: идентификатор пользователя, последним обновившего запись
- `is_active`: флаг активности записи (true/false)
- `version`: версия записи для контроля параллельных изменений
