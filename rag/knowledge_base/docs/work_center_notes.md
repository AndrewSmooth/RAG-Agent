# Table: work_center
## Notes
- Рабочий центр представляет собой локальную зону на производственной линии
- Каждый рабочий центр должен быть привязан к одной производственной линии
- Поле `code` должно быть уникальным в пределах одной линии
- Цикловое время используется для расчета производственной мощности
- Время переналадки критично при планировании производства нескольких продуктов
- Процент брака и коэффициент выхода годной продукции используются для анализа качества
- Данные о производительности анализируются регулярно для выявления узких мест
- Требования к обучению персонала обновляются при изменении технологий или оборудования
