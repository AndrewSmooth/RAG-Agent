# Table: work_order
## Notes
- Наряд-заказ представляет собой задание на выполнение конкретной операции в производственном процессе
- Каждый наряд связан с производственным заказом и определенной операцией
- Планируемая продолжительность и трудозатраты используются для составления расписания
- Фактические данные используются для анализа эффективности и контроля затрат
- Контроль качества может быть обязательным для определенных операций
- Переделка может потребоваться при обнаружении дефектов или несоответствий
- Время простоя и сверхурочные часы используются для анализа производственной эффективности
- Данные о стоимости материалов и трудозатрат используются для калькуляции себестоимости продукции
